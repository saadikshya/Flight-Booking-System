# Flight Booking System

## Overview
A web-based flight booking system developed as a student project that allows users to search, book, and manage flight reservations. The system includes full CRUD functionality with a responsive and user-friendly interface, designed to simulate a real-world booking experience.

## Features
- Flight search and booking interface
- User-friendly responsive UI
- CRUD operations for managing data
- Backend integration with database

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL

## How to Run
1. Clone the repository
2. Run on local server (XAMPP or similar)
3. Import database.sql
4. Open in browser
